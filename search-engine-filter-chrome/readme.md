# 功能

- 百度搜索默认过滤掉脚本之家(www.jb51.net ) 的内容
- 添加一个输入框，输入指定的url，使得在搜索时，来屏蔽掉来自特定地址的内容
- 添加完成后点击过滤按钮进行提示
